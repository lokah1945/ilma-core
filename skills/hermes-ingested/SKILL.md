## Memory System API

### Core Operations

| Operation | Method | Description |
|-----------|--------|-------------|
| Store | `await memory.store(key, value)` | Store a value with key |
| Retrieve | `value = await memory.retrieve(key)` | Retrieve by key |
| Delete | `await memory.delete(key)` | Delete a key |
| List | `keys = await memory.list_keys()` | List all keys |
| Search | `results = await memory.search(query)` | Semantic search |

### Configuration

```yaml
memory:
  backend: redis  # or: sqlite, memory
  host: localhost
  port: 6379
  ttl: 3600  # seconds
```

## Tool Registry API

### Registration

```python
from hermes.tools import ToolRegistry

registry = ToolRegistry()

@registry.register(name="my_tool", version="1.0.0")
async def my_tool(context):
    """My custom tool implementation."""
    return {"result": "success"}
```

### Discovery

```python
# Find tools by capability
tools = registry.find(capability="text-generation")

# Get tool by name
tool = registry.get("my_tool")

# List all registered tools
all_tools = registry.list_all()
```
